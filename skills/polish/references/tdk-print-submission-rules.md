# TDK Print Submission Rules Profile

Source basis:
- TDK's publication submission page defines formal expectations for works submitted for publication, including Word-compatible files, typography, paragraph indentation, title hierarchy, page numbering, footnotes, references, summaries, and keywords.

Primary reference:
- https://tdk.gov.tr/yayinlar/yayinlar-yayinlar/basilmak-uzere-verilecek-eserlerde-uyulmasi-gerekli-kurallar/

## TDK Academic / Institutional Profile
Use this profile when `book_mode.profile = "tdk_submission"` or when the user requests TDK/institutional formatting.

Mandatory profile expectations:
- Word or Word-compatible document.
- A4 page assumption.
- Times New Roman typography.
- 11-point main text.
- Full line spacing.
- 1.25 cm paragraph first-line indent.
- Main headings: 12 point, bold, uppercase, left aligned.
- Subheadings: 11 point, bold, only first letters uppercase, left aligned.
- Spacing between headings and paragraphs.
- Footnotes: 9 point, full spacing, justified, 0.5 cm paragraph indent.
- References section starts on a new page and includes only works cited in the text.
- Images, tables, charts, and figure captions use smaller caption typography.
- Non-Latin script sections must be deliberately placed and font requirements recorded.
- If the work requires abstracts, Turkish and English abstracts plus keywords must be present.

## Book / Trade Profile
Use the existing A5 or e-book profile for commercial novels and stories. Do not force TDK academic typography onto trade fiction unless requested.

## Blocking Conditions
- Claiming `tdk_submission` readiness without required typography/layout metadata.
- Missing references section in academic/research works with citations.
- Missing abstracts/keywords when the chosen profile requires them.
- Unmarked non-Latin script content.
