# Parity Gap Checklist

Use this checklist to compare current contracts vs upstream/original behavior expectations.

## Checklist
- [ ] Flow coverage parity (`propose/design/create/polish/rewrite`)
- [ ] Agent responsibility parity
- [ ] Required output artifact parity
- [ ] Verdict vocabulary parity
- [ ] Retry/boundary behavior parity
- [ ] Language policy parity with local product goals

## Required Report
- `{WORK_DIR}/_workspace/parity-gap-report.md`

Report must include:
- missing capabilities
- intentionally removed capabilities
- newly added capabilities
- compatibility risk notes
