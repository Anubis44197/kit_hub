## Editor-side readability baseline (book mode)

TDK does not define full modern editor UI layout behavior. For column-width and spacing accessibility in digital reading surfaces, use W3C references:

- WCAG visual presentation understanding: https://w3c.github.io/wcag/understanding/visual-presentation
- WCAG technique C20 (line width control): https://www.w3.org/WAI/WCAG20/Techniques/css/C20
- WCAG text spacing understanding: https://www.w3.org/WAI/WCAG21/Understanding/text-spacing.html
