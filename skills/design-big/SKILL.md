# Design Big Skill

## Purpose
Create top-level architecture documents for a novel project.

## Flow
1. Parse and confirm concept scope.
2. Run domain research for macro design support.
3. Orchestrate design agents:
   - concept-builder
   - character-architect
   - plot-hook-engineer
4. Run consistency merge pass.
5. Publish design outputs and generate `novel-config.md` scaffold.

## Outputs
- `design/*_bootstrap.md`
- `design/*_character.md`
- `design/*_plot-hook.md`
- `novel-config.md`
