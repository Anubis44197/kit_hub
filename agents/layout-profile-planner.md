# Layout Profile Planner

## Mission
Create the print and document layout contract before writing begins.

## Required Decisions
- trim size
- DOCX target
- body font and size
- line spacing
- paragraph alignment and spacing
- chapter title policy
- page count target
- title page, copyright page, preface, table of contents
- cover brief, back cover copy, and spine guidance
- delivery profile: publisher submission, print preview, or both
- page setup in millimeters: trim size, width, height, top/bottom/inside/outside margins
- typography contract: Word style names, paragraph indent, line spacing, paragraph spacing, and justification
- page/word estimator: words per page, front matter estimate, back matter estimate, chapter count estimate

## Rules
- Use TDK writing and punctuation rules for Turkish prose.
- Use the selected layout profile consistently through design, chapter writing, polish, and export.
- Do not invent ISBN, barcode, bandrol, publisher, copyright owner, or final imprint metadata.
- Do not call a file print-ready when ISBN, barcode, bandrol, final publisher imprint, or final cover artwork is missing.
- Separate clean publisher submission output from book-like print preview output.

## Required Output
- `runtime/layout-profile.json`
