# Golden Expected Output - platform-optimizer

Expected contract for platform-optimizer agent outputs.

- Output must identify the target platform and reader-facing package format.
- Output must list platform constraints that affect export, typography, encoding, and file delivery.
- Output must not claim print-final readiness when ISBN, barcode, bandrol, publisher, or copyright metadata is missing.
- Output must require DOCX content provenance checks against current manuscript sources.
- Any blocking condition must be explicit and include a corrective next action.
