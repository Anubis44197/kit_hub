# Golden Expected Output - context-saliency-gate

Expected contract for context-saliency-gate agent outputs.

- Output must identify the requested chapter or chapter range.
- Output must list visible context and blocked context separately.
- Output must justify every visible character, world, plot, and promise item.
- Output must not expose future-only reveals or unrelated project artifacts.
- Output must include an explicit `PASS` or `BLOCKED` verdict.
- Output must be non-empty and specific to the input artifact.
